# Architecture

This repository uses the standard GitHub profile repository layout.

## Layout

```text
.
├── README.md
├── AGENTS.md
├── CLAUDE.md
└── ARCHITECTURE.md
```

## Purpose

- `README.md` is rendered on the public GitHub profile page.
- `AGENTS.md` keeps repository-specific working rules.
- `CLAUDE.md` documents how Claude Code should handle this repository.
- `ARCHITECTURE.md` records the repository structure.

## Constraints

- No application source code.
- No build system.
- No generated assets unless explicitly needed for the profile.
- No secrets, private notes, or local machine paths.
